# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

**Projeto:** Redução dos Custos em Farmácias com AWS
**Data:** 28/02/2026
**Empresa:** Abstergo Industries (estudo de caso)
**Responsável:** Gleidson Fagno

---

## Introdução

Este relatório apresenta o processo de implementação de serviços da **AWS (Amazon Web Services)** na empresa **Abstergo Industries**, com foco na **redução imediata de custos operacionais** no setor farmacêutico. O projeto foi desenvolvido como atividade prática, visando aplicar conceitos de computação em nuvem, escalabilidade e otimização de recursos.

O objetivo principal foi selecionar **três serviços AWS** capazes de reduzir despesas com infraestrutura, aumentar a eficiência operacional e melhorar a disponibilidade dos sistemas.

---

## Descrição do Projeto

O projeto de implementação foi dividido em **três etapas**, cada uma abordando um serviço AWS estratégico para redução de custos.

### Etapa 1 – Amazon EC2 com Auto Scaling

* **Serviço:** Amazon EC2 (Elastic Compute Cloud)
* **Foco:** Redução de custos com servidores físicos
* **Descrição do caso de uso:**
  A empresa utilizava servidores físicos locais com alto custo de manutenção e baixa flexibilidade. Com a migração para instâncias EC2 e uso de **Auto Scaling**, os recursos computacionais passaram a ser ajustados automaticamente conforme a demanda, evitando desperdício e reduzindo custos operacionais.

---

### Etapa 2 – Amazon S3 para Armazenamento

* **Serviço:** Amazon S3 (Simple Storage Service)
* **Foco:** Armazenamento de dados com menor custo
* **Descrição do caso de uso:**
  Documentos fiscais, relatórios e backups passaram a ser armazenados no Amazon S3, eliminando a necessidade de servidores dedicados para armazenamento. O modelo de pagamento conforme o uso permitiu uma redução significativa de custos, além de aumentar a segurança e durabilidade dos dados.

---

### Etapa 3 – Amazon RDS

* **Serviço:** Amazon RDS (Relational Database Service)
* **Foco:** Otimização e redução de custos com banco de dados
* **Descrição do caso de uso:**
  O banco de dados local foi migrado para o Amazon RDS, removendo custos com manutenção, backups manuais e atualizações. O serviço gerenciado garantiu maior disponibilidade, escalabilidade e previsibilidade de custos, essenciais para o controle de estoque e vendas das farmácias.

---

## Conclusão

A implementação dos serviços AWS na empresa **Abstergo Industries** resultou em **redução de custos com infraestrutura**, maior eficiência operacional e melhoria na confiabilidade dos sistemas. A adoção de soluções gerenciadas permitiu que a equipe focasse no negócio principal, em vez de tarefas operacionais.

Recomenda-se a continuidade do uso dos serviços implementados e a avaliação de novas soluções em nuvem, como AWS Lambda e CloudWatch, para otimizar ainda mais os processos e custos da empresa.
